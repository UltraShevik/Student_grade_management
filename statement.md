## 1. Problem Statement

Managing student grades and records using paper or spreadsheets is inefficient and time-consuming for teachers and small institutions. There is a need for a fast, simple, and reliable digital tool to automate basic student record keeping.

---

## 2. Scope of the Project

This is a **command-line application (CLI)** built in Python that stores student records (name and grade) **in-memory** using a List of Tuples. The project focuses on core **CRUD** (Create, Read, Update, Delete) operations, menu navigation, and essential input validation (like preventing duplicate names).

---

## 3. Target Users

* **Teachers / Tutors:** Individuals managing grades for a single class or small group.
* **School Administrators:** Staff needing basic, quick access to student data.
* **Small Schools / Coaching Centers:** Institutions without large, complex database systems.

---

## 4. High-Level Features

* **Menu Interface:** Simple numbered menu for all operations (1-6).
* **Data Integrity:** Prevents the addition of duplicate student names.
* **Full CRUD:** Functions to **Add**, **Show**, **Search**, **Update**, and **Delete** student records.
* **Usability:** Features case-insensitive searching for better user experience.
